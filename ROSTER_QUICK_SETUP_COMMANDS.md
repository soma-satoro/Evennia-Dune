## Setup Instructions

For each character:

1. **Puppet the character:**
   ```
   @ic <Character Name>
   ```

2. **Copy and paste the entire command block** for that character (below)

3. **Exit back to your character:**
   ```
   @ooc
   ```

4. **Verify:**
   ```
   @ic <Character Name>
   +sheet
   @ooc
   ```

## Quick Checklist

After setting up each character, verify:

```
@ic <Character Name>
+sheet
```

Expected results:
- ✅ Skills set correctly (sum should match character level)
- ✅ 4-8 Focuses listed
- ✅ 5-7 Talents listed
- ✅ Drives rated 8, 7, 6, 5, 4 (one of each)
- ✅ Drive statements for drives rated 6+
- ✅ Biography sections filled in
- ✅ Languages configured
- ✅ Determination at 3

## Troubleshooting

**Commands not working?**
- Make sure you're puppeting the character first (`@ic <name>`)
- Check you have proper permissions (Builder+)

**Biography text too long?**
- The `+bio` command has character limits
- Break very long text into shorter segments

**Can't add language?**
- Check the language exists in your game
- Use `+language/list` to see available languages

**Drives not saving?**
- Make sure ratings are 8, 7, 6, 5, 4 (one of each)
- Drive statements required for 6+
